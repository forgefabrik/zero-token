# Zero-Token TTS Runtime

Dieses Verzeichnis enthält Manifest, Build-Workflow und Laufzeitquellen für die Zero-Token-TTS-Erweiterung.

Die VSIX bleibt klein. Beim ersten Start erkennt der Einrichtungsassistent Betriebssystem, Linux-Distribution und CPU-Architektur und lädt das passende versionsgebundene Runtime-Paket aus den GitHub-Releases dieses Repositorys.
